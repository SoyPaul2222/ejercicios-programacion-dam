                                       Llista de comandes i les seves descripcions

        1. $ mkdir carpeta\
        Descripció: Crea una carpeta anomenada "carpeta".
        
        2. $ cd carpeta\
        Descripció: Canvia de directori a "carpeta".
        
        3. $ cat fitxer\
        Descripció: Mostra el contingut d'un fitxer.
        
        4. $ gedit fitxer &
        Descripció: Crea o edita el "fitxer" en un editor de text.
        
        5. $ ls
        Descripció: Mostra una llista de carpetes o arxius.
        
        6. $ rm fitxer.
        Descripció: Elimina un fitxer que es diu "fitxer".
        
        7. $ git init
        Descripció: Permet inicialitzar un repositori git.
        
        8. $ git clone
        Descripció: Permet descarregar un repositori.
        
        9. $ git status
        Descripció: Comprova l'estat del git.
        
        10. $ git add
        Descripció: Afegeix un fitxer o arxiu en el repositori git.
        
        11. $ git commit
        Descripció: Aplica canvis.
        
        12. $ git push
        Descripció: Permet pujar els arxius o directoris a la branca del nostre repositori.
        
        13. $ git pull
        Descripció: Permet actualizar o fer canvis de un repositori remot.
