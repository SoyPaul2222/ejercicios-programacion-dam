# ejercicios-programacion-dam
Este repositorio recopila las actividades, ejercicios prácticos y notas teóricas que he ido desarrollando durante mis estudios para consolidar mi lógica de programación y dominar las bases del desarrollo de software.

---

## 🛠️ Tecnologías y Conceptos Practicados

* **Lenguaje Principal:** Java (Programación Orientada a Objetos).
* **Conceptos Clave:**
  * Sintaxis fundamental y tipos de datos.
  * Estructuras de control de flujo (bucles `for`, `while`, condicionales `if-else`, `switch`).
  * Estructuras de datos lineales (arrays y matrices).
  * Diseño de funciones y métodos modulares.
  * Introducción a clases y objetos.

---

## 📁 Contenido del Repositorio

* **Ejercicios Prácticos (`.java`):** Carpetas independientes con retos de código y lógica ordenados por temáticas.
* **Notas Teóricas (`.rst`):** Algunos archivos breves en formato reStructuredText que sirven como resúmenes y apuntes teóricos de soporte.

---

## 🧠 Sobre los Ejercicios

* La gran mayoría de las actividades de lógica están **completamente resueltas y funcionales**.
* Algunos retos avanzados se encuentran **parcialmente completados**, manteniéndolos aquí para documentar mi enfoque lógico e identificar los desafíos técnicos en los que sigo trabajando para mejorar día a día.
